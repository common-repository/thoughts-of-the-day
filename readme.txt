This Plugin is to manage and show daily thought, admin can add thoughts in each line for each day and will show the each line according to day.

== Description ==

= Requirements =

* WordPress 3.+ or 3.5 with WordPress plugin

== Installation ==

1. Download thoughts-of-the-day.zip
2. Extract and upload thoughts-of-the-day directory to the plugins/ directory
3. Enable Thought Of The Day plugin in the Plugin admin panel

== How To Use == 
use shortcode : [THOUGHTSOFTHEDAY]
Function : <?php echo do_shortcode( '[THOUGHTSOFTHEDAY]' ) ?>

== Frequently Asked Questions ==
= How we will add the thoughts ? =
In admin section there are thoughts textarea, Please add each thought in each line.
= How many thoughts should be add ? =
31 thoughts only.
